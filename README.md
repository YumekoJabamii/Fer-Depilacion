# Fer Depilación

Sitio web para **Fer Depilación**, centro especializado en depilación con cera y definitiva ubicado en Barracas, Buenos Aires.

## 🌐 Ver el sitio

El sitio está publicado en: `https://<tu-usuario>.github.io/<nombre-del-repo>/`

## 📁 Estructura

```
/
└── index.html   # Sitio completo (HTML + CSS + JS en un solo archivo)
```

## 🚀 Publicar en GitHub Pages

1. Subí este repositorio a GitHub
2. Andá a **Settings → Pages**
3. En *Branch*, seleccioná `main` y carpeta `/ (root)`
4. Guardá — en unos minutos el sitio estará en vivo
